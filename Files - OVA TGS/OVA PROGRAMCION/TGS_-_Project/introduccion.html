Introduccion

Actividad

Este curso esta diseñado para hacer un acercamiento a una de las herramientas mas importantes del mundo actual, la programacion. Para iniciar este curso se han de tener ciertos conocimientos previos como son:

  • Conceptos de informática y computación
  • Matemáticas básicas
  • Lógica booleana
  • Manejo de internet

 

Al finalizar el curso se espera que el estudiante:

  • Pueda diseñar e implementar algoritmos que permitan la solución de problemas de baja complejidad.
  • Conozca los conceptos básicos para el manejo del lenguaje.
  • Haga uso de las estructuras repetitivas, de condicionales y de las operaciones lógicas necesarias para resolver los problemas que se vayan presentando a lo largo del curso.
